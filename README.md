### 1. 🚗 Uber Data Analysis Dashboard

**File:** `Uber project.pbix`

A comprehensive Power BI dashboard analyzing Uber ride data to uncover insights about:
- 🗺️ Ride patterns and trends

- Interactive visualizations with drill-down capabilities
- Time-series analysis of ride patterns

- Analyze geographic hotspots for service expansion
- Track performance metrics and revenue trends
- Support data-driven decision making

## 📥 Getting Started

### Prerequisites

Before you can open and explore these Power BI projects, ensure you have:

- **Microsoft Power BI Desktop** installed on your system
  - Download from: [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
  - Supported OS: Windows 10 or later
  - Free version available for personal use

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/xainy75/Power-BI-projects-.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Power-BI-projects-
   ```

3. **Open Power BI Desktop**
   - Launch the Power BI Desktop application

4. **Open a project file**
   - File → Open → Browse to the `.pbix` file
   - Or double-click the `.pbix` file to open directly

## 💡 How to Use

### Opening Projects

1. **Double-click** the `.pbix` file, or
2. Open Power BI Desktop → **File** → **Open** → Select the `.pbix` file

### Exploring Dashboards

- 🖱️ **Click** on visualizations to cross-filter other visuals
- 🔍 **Hover** over data points to see detailed tooltips
- 📊 **Use slicers** to filter data by different dimensions
- 📅 **Select date ranges** to analyze specific time periods
- ⬇️ **Drill down** in hierarchies for deeper insights

### Customizing Reports

1. Switch to **Edit mode** in Power BI Desktop
2. Modify visualizations, add new pages, or create custom measures
3. Update data sources to connect to your own data
4. Save your customized version
